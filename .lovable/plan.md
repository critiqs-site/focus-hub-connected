# AI Chat - Full Todo Management (COMPLETED)

## Features Implemented
1. **Non-streaming AI** — instant JSON responses
2. **List todos** — AI sees full todo context
3. **Delete todos** — `[ACTION:DELETE:id:text]`
4. **Rename todos** — `[ACTION:RENAME:id:newText:oldText]`
5. **Transfer todos** — `[ACTION:TRANSFER:id:dividerId:text:sectionName]`
6. **Change icons** — `[ACTION:ICON:id:iconName:text]`
7. **Suggest todos** — `[ACTION:SUGGEST:dividerName:text:icon]`
8. **Add All** — `[ACTION:ADD_ALL]`
